## Sobre o Método ADVC

O Método ADVC (Arquiteto de Valor Cloud) é uma jornada prática focada em AWS, com aulas, demonstrações, laboratórios e desafios que simulam o dia a dia real de cloud — com o objetivo de formar alguém que sabe **projetar, implementar e defender soluções** com segurança.

## Propósito deste repositório

A ideia aqui é simples e direta: **centralizar minha evolução**.

Tudo o que eu pratico no método fica versionado nesse repositório — labs realizados, desafios concluídos e materiais organizados — para servir como histórico técnico, referência de estudo e prova de consistência na prática.
