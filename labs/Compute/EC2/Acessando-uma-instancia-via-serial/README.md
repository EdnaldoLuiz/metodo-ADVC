<h1 align=center> Amazon EC2 - Acessando uma instância via serial </h1>

<h2> Console serial EC2 </h2>

Configurar o acesso ao console serial de instâncias EC2 é um processo fundamental para a administração eficaz e a resolução de problemas em ambientes de nuvem AWS. Ao estabelecer esse acesso, os administradores ganham a capacidade de interagir com as instâncias de forma segura, mesmo em situações em que a conectividade via SSH pode estar comprometida. Essa configuração é especialmente valiosa para cenários de solução de problemas, recuperação de instâncias ou alterações nas configurações de rede. O acesso ao console serial oferece uma camada adicional de controle e é uma prática recomendada para assegurar a continuidade operacional e a manutenção proativa dos recursos na nuvem.

<h2> Conteúdo do laboratório </h2>

Neste laboratório, você irá aprender a criar uma instância EC2 e a utilizar o AWS Instance Connect por meio de conexões serial e SSH. Além disso, aprenderá a utilizar a conexão SSH para criar uma senha para o usuário, que será necessária para a conexão via serial. 

<h2>Tarefas a serem executadas</h2>

1. Realizar o login na conta AWS
2. Criar instância EC2
3. Realizar acesso a instância vai EC2 Serial Console
4. Realizar acesso  a instância via instance Connect para criação de senha
5. Validado sua senha de acesso sua EC2 no Console de série do EC2
